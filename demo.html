Placeholder demo page

Demo page for placeholder jQuery plugin, includes examples of various ways to use the placeholder component.

See project page for documentation.

Basic example

Markup

<textarea placeholder="Write something here..."></textarea>

Hide on focus gets

Markup

<textarea data-placeholder-hide-on-focus="true" placeholder="Write something here..."></textarea>

Elaborate animations

Markup+CSS

<textarea placeholder="Write something here..."></textarea>

.elaborate-animation .jquery-placeholder-text {
    margin-left: 0px;
}

.elaborate-animation .jquery-placeholder-text.jquery-placeholder-text-focused {
    margin-left: 100px;
    font-style: italic;
}

Right-to-left

Markup+CSS

<textarea placeholder="Write something here..."></textarea>

.rtl .jquery-placeholder-text {
    direction: rtl;
}

.rtl textarea {
    direction: rtl;
}

Z-index robustness

The placeholder texts do not show through the overlay, no matter what - there is no reliance on some specific z-index value.

Markup+CSS

.input-placeholders .jquery-placeholder-text {

}

Base CSS

The base css that is applied to every placeholder with low specificity.

.jquery-placeholder-text {
    color: #aaa;
    -moz-transition: color, margin-left 0.15s ease-in;
    -webkit-transition: color, margin-left 0.15s ease-in;
    -o-transition: color, margin-left 0.15s ease-in;
    transition: color, margin-left 0.15s ease-in;
}

.jquery-placeholder-text.jquery-placeholder-text-focused {
    color: #ddd;
}
